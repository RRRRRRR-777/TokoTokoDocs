## TokoTokoDocs
TokoTokoのドキュメント用リポジトリ
## 開発用リポジトリ 「[TokoToko](https://github.com/RRRRRRR-777/TokoToko)」
* 「とことこ - おさんぽSNS」のリポジトリ
* このアプリは、「日常の散歩体験を、友人や家族と手軽に共有する」ことを目的としたSNS的な機能を持つサービスです。
* ただ歩くだけの時間を“楽しい体験”に変え、その記録を共有することで、新たな発見や会話が生まれることを目指します。
